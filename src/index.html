📖 쿠키 추출 방법
방법 A — Chrome DevTools
1. 기존 계정으로 youtube.com 로그인
2. F12 → Application 탭
3. 좌측 Cookies → https://www.youtube.com 클릭
4. 쿠키 목록 전체 선택 후 복사

방법 B — EditThisCookie 확장 (권장)
1. Chrome에 EditThisCookie 확장 설치
2. YouTube 로그인 후 확장 클릭
3. Export (JSON) 버튼 클릭 → 복사
4. 위 입력란에 붙여넣기
